What these values mean:

  • MSE: Measures the average squared difference between predicted and actual values. Lower values are better.
  • RMSE: Similar to MSE but in the same units as the target variable. Lower values indicate better performance. A good RMSE for life expectancy predictions would generally be between 0-5 years.
  • R²: Explains how much variance in the target variable is explained by the model. Values range from 0 to 1, with closer to 1 indicating better performance. An R² above 0.8 is usually considered excellent.
